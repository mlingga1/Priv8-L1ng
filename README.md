## Installation for Termux

```sh
$ apt update
$ apt upgrade
$ apt install python
$ apt install git
$ git clone https://github.com/mlingga1/Priv8-L1ng
$ cd Priv8-L1ng
$ python -m pip install -r requirements.txt
$ python bot.py
```

## Installation for VPS

```sh
$ git clone https://github.com/mlingga1/Priv8-L1ng
$ cd Priv8-L1ng
$ python3 -m pip install -r requirements.txt
$ python3 bot.py
```
